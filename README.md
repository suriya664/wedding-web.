# Wedding Website

A beautiful, responsive wedding website built with HTML, CSS, JavaScript, jQuery, AJAX, and Bootstrap.

## Features

- **27 Complete Pages**: All pages including homepage, about, events, gallery, RSVP, and more
- **Responsive Design**: Fully responsive for mobile, tablet, and desktop devices
- **Modern UI**: Beautiful design with Bootstrap 5 and custom CSS
- **Interactive Elements**: Forms with AJAX submission, smooth scrolling, and animations
- **Consistent Navigation**: Header and footer on all pages with active page highlighting
- **Image Optimization**: Lazy loading for better performance
- **Social Media Integration**: Social icons with hover effects

## Setup Instructions

1. All files are ready to use
2. Add a favicon.ico file to the `images/` folder
3. Open `index.html` in a web browser to view the website
4. For form submissions, configure your backend endpoint in the JavaScript file

## Customization

- Update contact information in the footer of all pages
- Modify wedding date in the countdown timer (index.html)
- Replace placeholder images with your own
- Customize colors in `css/style.css` (CSS variables)
- Update social media links in the footer
